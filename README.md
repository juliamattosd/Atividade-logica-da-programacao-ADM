# Atividade-logica-da-programacao-ADM
##  Sobre o Projeto
Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de lógica de programação utilizando Python no Google Colab. Os exercícios abordam situações práticas, trabalhando com listas, estruturas condicionais e repetição, além de estimular o raciocínio lógico.

##  Descrição das Questões

###  Questão 1 – Classificação de Temperaturas  
Nesta questão, uma lista de temperaturas é analisada e cada valor é classificado em categorias como "frio", "agradável" ou "quente", utilizando estruturas condicionais.

###  Questão 2 – Exibição e Contagem  
O programa exibe cada temperatura com sua classificação e também conta quantas vezes cada categoria aparece, utilizando loops e contadores.

###  Questão 3 – Análise de Notas  
Uma lista de notas é utilizada para classificar alunos como aprovados ou reprovados, aplicando regras condicionais.

###  Questão 4 – Cálculos com Valores  
O programa realiza operações com uma lista de valores, como soma total, identificação de valores acima de um limite ou aplicação de descontos.


##  Explicação da Lógica

A lógica utilizada segue um padrão:

- Uso de listas para armazenar dados  
- Estruturas de repetição (`for`) para percorrer os dados  
- Estruturas condicionais (`if`, `elif`, `else`) para decisões  
- Contadores e acumuladores para análises  
- Exibição dos resultados de forma clara
